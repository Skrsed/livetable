### LIVETABLE APP

#### Desc:
- Event driven table with live updating data

#### Functionality
- Edit table header based on set of records keys
- CRUD records

#### How to run:
- `docker-compose up -d --build` run localy
- `docker-compose -f docker-compose.yml up -d --build` run with prod env (ci only)
